Steps to Install Java Language

1. Environment for Java programming

The terminal command prompt is pre-installed in your LINUX operating system. Our task now is to install the jdk using terminal to execute the Java programs.

2. How to install Jdk using terminal in Linux?This step-by-step guide will show you how to install Java on Ubuntu 18.04, 20.04, or 22.04.

You can find the terminal icon on your Desktop screen. If not, you can find it through the search menu. A shortcut to open terminal is: Press Ctrl + Alt + T. Open terminal so that we can proceed with the further steps to install the GCC Compiler. Next, install the latest Java Development Kit by running the following command:

sudo apt install default-jdk       

Type ‘y’ when the command prompt asks “Do you want to continue?” and then press Enter. Thereafter, your installation will begin.

3. How to check the installed compiler version?

You can find the terminal icon on your Desktop screen or on the search menu. A shortcut to open terminal is: Press Ctrl + Alt + T simultaneously. Now, verify whether the software is part of your system. Check the version of Java running on your system using the following command:

JAVA -- version       

This version is 17.0.6. If you don’t see such a message on your display screen, then it probably means that you haven’t successfully installed the JAVA Development kit.

4. How to create a Java program?

will show you how you can execute Java directly from your favorite terminal window. Fear not! The procedure is quite easy, and after reading the entire article you should be able to run your own Java code in the terminal..

In order to locate your files on the Desktop folder and to change the reference of the current directory to Desktop, use this command:
cd Desktop       

Key takeaway: Commands are case sensitive. The command for creating a program in Java is:

Helloworld.java

Open this file and write a basic code – “Hello World!”

5. How to compile and run a Java program?

Now that we have created a java program called Hello.java, it is now time to compile it and run it in terminal. To compile the code, we use the "javac" command: javac filename

Here, the ‘javac’ command is followed by the file name with java extension, which is, ‘program’ in this case. Key takeaway: The executable file name can be different from the source file name. In order to run the program, use the command: after the successful completion of javac command hit java "filename"...........
